Difference between functions n and note

使用 synth 時，n 與 note 功能完全相同，下面的範例會播放 C 、E (比 C 高 4 個半音) 及 G (比 C 高 7 個半音) 三個音：

使用 sample 時，n 是指 sample 資料夾中的檔案順序，依字母順序（升序），並從 0 開始。這時使用 note 時，sample 的音調會被調高或調低（同時 sample 的時長也會受到影響）。
